# ev

## 项目介绍

ev是一个综合性Web应用平台，提供多样化的工具和功能模块，帮助用户高效完成日常任务。该平台采用现代化前端技术栈，确保良好的用户体验和跨浏览器兼容性。

## 技术栈

- **前端框架**：HTML5, CSS3, JavaScript
- **样式框架**：Tailwind CSS
- **图标库**：Font Awesome
- **环境管理**：自定义环境配置系统
- **存储**：localStorage

## 项目结构

```
static/
├── js/                  # JavaScript文件目录
│   └── environment.js   # 环境配置管理
├── module/
│   └── index/           # 功能模块目录
│       ├── contact.html     # 联系页面
│       ├── qrcode_page.html # 二维码页面
│       ├── buisinessCard.html # 名片展示系统
│       ├── 2048.html        # 2048游戏
│       └── ...              # 其他功能模块
├── index.html           # 首页
├── README.md            # 项目文档
└── .gitignore           # Git忽略配置
```

## 核心功能

- **重要日子记录**：记录和展示重要日期及倒计时
- **二维码生成**：生成各种类型的二维码，支持自定义内容
- **名片展示系统**：展示用户名片信息和相关图片
- **2048游戏**：经典2048游戏实现
- **矩阵工具**：矩阵相关计算和动画效果
- **邮件工具**：邮件相关功能和管理
- **50OK平台**：专业的游戏娱乐平台

> 注意：所有工具模块已整合到核心功能模块中，提供统一的访问体验
- **环境切换**：支持开发/生产环境切换，配置保存在localStorage

## 安装与使用

### 开发环境

1. 克隆或下载项目到本地
2. 在项目根目录启动HTTP服务器
   ```bash
   python -m http.server 8000
   ```
3. 浏览器访问 `http://localhost:8000`

### 环境配置

系统支持自动环境检测和手动切换：
- 可通过URL参数 `?env=development` 切换到开发环境
- 点击导航栏环境切换按钮手动切换环境
- 环境配置会保存在localStorage中

## 开发指南

### 添加新功能模块

1. 在 `module/index/` 目录下创建新的HTML文件
2. 在首页添加对应的导航链接
3. 遵循项目的代码规范和设计风格

### 代码规范

- **HTML**：使用语义化标签，保持良好缩进
- **CSS**：优先使用Tailwind CSS类，避免内联样式
- **JavaScript**：使用ES6+语法，保持代码简洁
- **命名**：使用有意义的变量和函数名，遵循驼峰命名法
- **注释**：添加必要的注释，解释复杂逻辑

## 项目维护

### 版本管理

遵循语义化版本控制规范：
- **主版本号**：不兼容的API变更
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正
